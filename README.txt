Due to Run this project we need to setup bunch of files 
0. Make sure You have NODE package manager(NPM) installed on your system 
	(download from here https://nodejs.org/en/download/)

1. Open a Terminal and Run <npm install> in the root folder of project

2. Run Your Local SqlServer Server and Remember username, password and 
	the port which sql is running on .

3. Open app.js file and go to //Create Connection

4. In this section change the username, password and host as your local Sql-server 

5. Open a Terminal and Go to the root folder of project

6. Run 'node app'

7. Open a Browser and go to 'localhost:3000'
